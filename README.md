### 2.8.1自动互助提示
编辑config下 → task_before.sh文件

内容如下

   ```diff
#!/usr/bin/env bash
if [[ $(ls $dir_code) ]]; then
    latest_log=$(ls -r $dir_code | head -1)
    . $dir_code/$latest_log
fi
   ```
